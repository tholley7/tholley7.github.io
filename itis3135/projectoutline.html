Project Assignment 1: Outline

This website would be the main page for a personal trainer, who goes by the name Strong Fitness. They are based out of a small town and their target audience is mainly middle-aged women in the area. The website will include the following pages:

  • Home
  • About
  • Programs
  • Contact

Site Map

Page Designs

Home

  • The Home page serves as the landing/splash page for the site.
  • The main audience of the Home page is potential clients.
  • Content
    • Hero Image
    • Call to Action
    • “Schedule Consultation” button
      • Leads to Contact page
    • Testimonials carousel

About

  • The About page describes the business owner and the history of the business as well as the trainer’s credentials.
  • The target audience of the About page is potential clients who were intrigued by the Home page and want to learn more about the proprietor.
  • Content
    • The trainer's headshot
    • A brief blurb describing the trainer’s background and experience
    • A brief description of the business and its offerings
    • A button that leads to the Programs page

Programs

  • The Programs page serves as an overview of the different types of training programs Strong Fitness offers.
  • The target audience is potential clients who wish to set up a consultation but need to get an idea of which program is best suited for them.
  • Content
    • Side-by-side titles and a bulleted breakdown
    • The time period of each program under its name
    • A button alongside each program that links to the Contact Form page and auto-fills the name of the program in the subject line
    • A divider line between each program element

Contact

  • The Contact page serves as a contact form to schedule a consultation.
  • The target audience for this potential clients who are impressed by the site and the trainer’s qualifications and wish to set up a time to discuss specifics and likely sign up for a class.
  • Content
    • Icon buttons that link to social accounts
    • Contact Form
      • First and Last Name - 2x Text Input
      • Sender’s Email - Email Input
      • Subject Line - Text Input
        • Pre-filled if the user clicked a button on the Programs page to navigate to this page.
      • Messsage Content - Text Area

Dynamic Functionality

The JavaScript elements with dynamic functionality on this site are:

  • Contact Form
    • Used on the Contact page
    • Used to allow potential clients to contact the trainer through the website
  • Image Gallery or Carousel
    • Used on the Home page for the display of testimonials
    • Used to show a rotating view of client review
